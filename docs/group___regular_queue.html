Detailed Description

Static memory FIFO queue for embedded systems.

This module provides a regular queue implementation that follows the First-In-First-Out (FIFO) principle. It's designed for embedded systems where predictable memory usage and no dynamic allocation are required.

Use Cases:
  • UART receive buffers
  • Task scheduling queues
  • Event processing systems
  • Command queues for protocols
  • Data logging buffers
